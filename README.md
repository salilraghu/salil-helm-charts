# salil-helm-charts

**Usage**

[Helm](https://helm.sh/) must be installed to use the charts. Please refer to Helmâ€™s [documentation](https://helm.sh/docs/) to get started.

Once Helm is set up properly, add the repository as follows:
```
helm repo add salilapp https://salilraghu.github.io/salil-helm-charts/
helm repo update

```
You can then run `helm search repo salilapp` to see the charts.
